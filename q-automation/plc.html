Series PLC Wecon

  • PLC Serie LX6V

    Máxima capacidad y alto rendimiento industrial

    La serie LX6V de Wecon está diseñada para entornos industriales exigentes: hasta 5 MB de memoria de programa, comunicación EtherCAT y USB Tipo C, y compatibilidad con Ladder, Instruction Table, FB y FC. Soporta expansiones modulares, alta velocidad de ejecución (~0.01-0.02 µs) y aplicaciones en robótica, maquinaria de envasado o sistemas críticos.

    • Memoria de hasta 5 MB de programa y 0.01 µs a 0.02 µs por instrucción.
    • EtherCAT incorporado + puerto USB Tipo C para configuración rápida.
    • Lenguajes de programación: Ladder, Instruction Table, FB, FC.
    • Expansión modular de E/S: hasta 32 módulos (y 64 con módulo de potencia adicional).
    Ver modelo en Wecon

  • PLC Serie LX5V

    Diseñado para velocidad, control de movimiento y precisión

    La serie LX5V combina gran velocidad de ejecución, múltiples ejes para control de pulsos y hasta 100 interrupciones de conteo de alta velocidad. Ideal para aplicaciones de automatización que requieren control dinámico, posicionamiento y sincronización en tiempo real.

    • Salida de pulsos de alta velocidad en 8 ejes simultáneos.
    • Tiempo de ejecución básico: 0.01 µs a 0.03 µs.
    • Soporte para comunicaciones de hasta 921.600 bps.
    • Compatible con módulos de expansión LX3V y sistemas modulares de alta velocidad.
    Ver modelo en Wecon

  • PLC Serie LX3VP

    Solución eficiente para sistemas industriales medianos

    La serie LX3VP ofrece un excelente equilibrio entre funcionalidad y coste: soporta programas de hasta 64 k steps, redes N:N entre PLCs (2-8 unidades), salidas de pulsos de hasta 200 kHz, expansión analógica/I/O y comunicación RS-485. Perfecta para control intermedio, integradores y sistemas con crecimiento previsto.

    • Capacidad de programa: hasta 64k steps.
    • Salidas de pulsos de alta velocidad: 200 kHz.
    • Red N:N intercambiable: interconexión de 2 a 8 PLCs.
    • Expansión analógica e I/O mediante múltiples módulos + opción de placa BD RS-485.
    Ver modelo en Wecon

PLC Serie A500

La solución premium todo-terreno para automatización global

La serie A500 de Wecon es la línea más avanzada para integradores industriales: ofrece redundancia de sistema, red EtherCAT doble, módulos de seguridad SIL3, sistema HMI integrado y conectividad IoT nativa. Diseñada para plantas grandes, infraestructura crítica y sistemas con alta demanda de tiempo de actividad.

  • Redundancia de CPU y fuente de alimentación para tolerancia de fallos.
  • Doble puerto EtherCAT, ProfiNet y Modbus TCP integrados.
  • Módulos de seguridad SIL3 / PL e para aplicaciones críticas.
  • Conectividad IoT nativa, IIoT y monitorización remota con nube.
